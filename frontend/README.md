# 쿠폰 시스템 프론트엔드

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 주요 기능

1. **쿠폰 목록**: 발급 가능한 쿠폰 조회 및 발급
2. **내 쿠폰**: 보유한 쿠폰 목록 조회 (사용 가능/사용 완료 필터)
3. **결제**: 쿠폰 선택 및 사용

## 환경 변수

백엔드 API는 `vite.config.ts`의 proxy 설정을 통해 `/api`로 프록시됩니다.
기본 백엔드 주소: `http://localhost:8080`

## 사용 방법

1. Member ID 입력 (기본값: 1)
2. 각 페이지에서 쿠폰 발급/조회/사용 가능
